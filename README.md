# Finance-Dashboard-using-Power-BI
A fully interactive Personal Finance Dashboard built in Power BI, featuring Income analytics, Expense tracking, Savings insights, Targets, and multi-year performance comparison.
Personal Finance Dashboard (Power BI)

This project is a complete Finance Analytics Dashboard developed in Power BI.
It helps users track Income, Expenses, Savings, Targets, and overall Net Worth, with monthly and yearly insights.
